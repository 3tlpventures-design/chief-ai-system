# Team Meetings

Notes from team meetings and 1:1s.

## Layout

| Folder | Contents | Filename |
|---|---|---|
| `<YYYY>/` (e.g. `2026/`) | Group meetings — standups, all-hands, project syncs | `YYYY-MM-DD-<short-slug>.md` |
| `one-on-ones/` | 1:1 logs, one file per person (long-running) | `<person-name>.md` |

Start a new year folder each January.

## Templates

- Group meetings: copy [`_template.md`](_template.md) into the appropriate year folder
- 1:1s: copy [`one-on-ones/_template.md`](one-on-ones/_template.md), rename to `<person-name>.md`, append new entries chronologically (newest at top)

## Conventions

- **Action items** — every meeting note ends with an Action Items section. Each item names an owner and a date. Unfinished items roll into the next meeting's prep.
- **Attendees** — list everyone present; mark `[remote]` next to anyone not in the room.
- **Decisions** — separate from action items. Decisions are conclusions; action items are work to do.
- **Don't transcribe** — capture decisions, action items, and surprises. Skip play-by-play.
